# 统一设置保存与即时应用设计

## 目标

设置页只保留一个“保存并应用”按钮。用户选择快捷键和识别模型后，应用一次性持久化这些选择，并在不重启进程的前提下切换到新配置。

## 方案

保存时先写入配置并创建候选 `AppHost`。候选宿主先加载 ASR 和标点模型，但不启动全局快捷键监听；原宿主在此期间继续工作。候选模型加载成功后，释放原宿主并启动候选监听器，随后将窗口、日志和录音浮窗订阅切换到候选宿主。模型加载失败时，候选宿主被释放，原宿主保持运行。

设置页中的“下载模型”仅负责安装文件；下载完成后提示用户点击统一保存按钮。保存期间禁用设置控件并展示进度状态，避免重复切换。

## 约束与验证

`VoxPen.Core` 仅提供平台无关的设置选择写入；SharpHook、ASR 实例和宿主切换保留在 App/Windows 层。为核心配置选择添加 xUnit 红绿测试；用完整 Core 测试和 App 构建验证。README 删除“快捷键/模型需要重启”的用户说明，改为说明通过设置页保存会立即应用。
